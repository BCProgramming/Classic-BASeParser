ABS Function

Returns the absolute value of a number.

Syntax:

ABS(value)

Or:

|Value|

Arguments

ArgumentDescription
ValueA numeric value to get the absolute value of.

Remarks

The Abs function returns the absolute value of a number, which is the positive magnitude of its difference from 0. The "Pipe" character, |, can be used to delimit a value. The ABS function is called on the expression contained within.

example:

The following script uses the abs function:

STORE(X,"null"); While(Not IsNumeric(X),(X:=INPUTBOX("Enter a Number:"));If(Not (IsNumeric(X)),Msgbox(Str(X) + " is not a number. Enter a number."),0)) msgbox("Absolute Value of " + Str(X) + " Is " + Str(Abs(X)))
