SeqEx Function

Syntax:

SeqEx(Expression,Varname,InitialValue,IncrementExpression,TerminateExpression,IterationLimit)

Arguments

ArgumentDescription
ExpressionExpression to execute. This expression is executed to acquire each successive value.
VarnameVariable name as it appears in Expression.
InitialValueStarting value of [i]Varname[/i]
IncrementExpressionExpression executed each iteration. Generally used to Store() a new value into Varname
TerminateExpression The Function will run until This expression is true. it cannot be a constant expression.
Iteration LimitMaximum number of iterations before the function is automatically stopped. negative numbers indicate to run until terminateexpression is true. Defaults to -1.

Remarks

The SeqEx Function is an extended version of the Seq Function that can be used for more complicated or dynamic Sequencing. However, it is far more dangerous- passing certain expressions can cause the function to execute infinitely until memory is exhausted for the creation of the return array.

Example

The following example creates an array of all the powers of 2 from 2^1 to 2^50: SEQEX(2^X,X,1,STORE(X,X+1),X>50) The expression, "STORE(X,X+1)" is executed each iteration, and the value of "2^X" is added to the return array until "X>50" is true. The return value is:
